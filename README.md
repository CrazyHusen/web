# web -- 此分支为响应式布局分支，相对稳定，不再维护

个人网站项目，包括博客、代码库、文件下载、留言、登录注册等功能

------------

## 主要文件结构如下

- `WebContent` -> 前端源文件 （前端三方插件在`/WebContent/plugins`下）
- `config` -> 配置文件
- `docs` -> 说明文档
- `libs` -> 后端jar包
- `src` -> 后端源文件

------------
